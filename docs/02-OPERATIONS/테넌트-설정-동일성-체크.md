## API 호출 흐름 (동일)

- 프론트(https://sswe.co.kr) → API(https://api.hakwonplus.com) 요청 시 `Origin: https://sswe.co.kr`, `X-Tenant-Code: sswe` 전송.
- 백엔드: CORS에서 `https://sswe.co.kr` 허용 → 200 + Access-Control-Allow-Origin 반환.
- tenant resolver: Host가 api.hakwonplus.com이면 X-Tenant-Code로 테넌트 결정 → sswe 인식.

## 배포 후 확인

- 백엔드 배포가 끝나면 CORS 에러 해소되어야 함.
- https://sswe.co.kr 접속 → /api/v1/core/program/ 200 → 로그인 페이지 표시.
