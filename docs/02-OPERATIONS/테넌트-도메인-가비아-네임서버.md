# 테넌트 도메인 연결 — 가비아 네임서버 (Cloudflare 연결용)

**도메인을 Cloudflare에 연결하려면** 가비아 도메인 관리에서 **네임서버를 Cloudflare에서 받은 값**으로 설정해야 한다.  
(가비아 기본 NS가 아니라 **Cloudflare zone 추가 시 부여된 NS 2개**를 등록해야 함.)

---

## 1. 누가 뭘 하는지

- **에이전트**: 새 도메인에 대해 Cloudflare zone 추가 후, 부여된 **네임서버 2개**를 운영자에게 전달.
- **운영자**: 가비아 [도메인 관리](https://domain.gabia.com) → 해당 도메인 → **네임서버 설정**에 1차·2차만 등록 후 **「등록했다」**라고 알림.

에이전트가 zone 추가 시 출력하는 1차·2차 값을 그대로 가비아에 넣으면 된다.

---

## 2. 가비아에 넣을 값 (예시)

Cloudflare zone 추가 스크립트 실행 시 출력되는 형식:

```
1차: xxx.ns.cloudflare.com
2차: yyy.ns.cloudflare.com
```

- 가비아 **네임서버 설정** 1차·2차에 위 두 개만 등록. 3차 이후는 비워두면 됨.
- 도메인마다 부여되는 NS가 다를 수 있으므로, **해당 도메인용 스크립트 출력값**을 사용할 것.

---

## 3. 네임서버 값 다시 확인하려면

이미 Cloudflare에 추가된 도메인들의 zone별 네임서버를 조회:

```powershell
cd backend
.\scripts\get-cloudflare-nameservers.ps1
```

`backend\.env`에 `CLOUDFLARE_EMAIL`, `CLOUDFLARE_API_KEY` 필요.
