# 테넌트 도메인 연결 — 가비아 네임서버

가비아(domain.gabia.com)에서 새 도메인(예: sswe.co.kr)을 등록·연결할 때 **네임서버 설정**에 넣을 값입니다.  
다른 테넌트 도메인(tchul.com, limglish.kr, ymath.co.kr 등)과 동일하게 가비아 기본 네임서버를 사용합니다.

## 네임서버 값 (도메인 관리 → 네임서버 설정)

| 구분 | 값 |
|------|-----|
| **1차** | `ns.gabia.co.kr` |
| **2차** | `ns1.gabia.co.kr` |
| **3차** | `ns.gabia.net` |

4차~13차는 비워두거나 "데이터 없음"으로 두면 됩니다.

## 적용 후

- 코드/DB에 테넌트·도메인을 추가한 뒤(`setup_three_tenants` 또는 관리 API)
- DNS 전파 후 해당 도메인으로 접속 시 해당 테넌트로 라우팅됩니다.
