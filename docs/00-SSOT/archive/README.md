# 00-SSOT 아카이브 (레거시·참고용)

**실행/배포에 사용하지 않습니다.** 정식은 [v1](../v1/)만 사용합니다.

| 하위 | 설명 |
|------|------|
| **v4/** | 이전 SSOT v4 문서·params·runbook·reports. 풀셋팅 v1 전환 후 참고용. |
| **v3/** | v3 문서·증명 (state-contract, RESOURCE-INVENTORY, CHANGELOG 등). |
| **legacy_reports/** | 과거 풀스택/드리프트/검증 리포트. |

유지보수·체킹 시 이 폴더만 보면 구버전 구성을 한눈에 파악할 수 있습니다.
