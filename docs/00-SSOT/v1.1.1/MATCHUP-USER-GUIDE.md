# AI 매치업 사용 가이드

## 매치업이란?

시험지나 문제집을 업로드하면 AI가 각 문제를 분석하고, 비슷한 유형의 문제를 자동으로 찾아주는 기능입니다.

---

## 기본 사용법

### 1. 문서 업로드

1. **자료실 > 매치업** 탭으로 이동합니다.
2. 좌측 상단 **+** 버튼 또는 **문서 업로드** 버튼을 클릭합니다.
3. 시험지 PDF 또는 이미지(PNG, JPG)를 선택합니다.
4. 제목, 과목, 학년을 입력합니다. (제목은 자동 채워짐)
5. **업로드** 버튼을 클릭합니다.

> **팁**: 선명하게 스캔한 시험지일수록 AI 인식률이 높습니다.

### 2. 문제 추출 확인

- 업로드 후 약 10~30초 내에 AI가 문제를 자동으로 추출합니다.
- 좌측 문서 목록에서 상태를 확인할 수 있습니다:
  - ⏳ 처리 중 — AI가 분석 중
  - ✅ N문제 — 분석 완료 (N개 문제 추출)
  - ❌ 실패 — 분석 실패 (재시도 가능)

### 3. 유사 문제 찾기

1. 좌측 문서 목록에서 **문서를 클릭**합니다.
2. 추출된 문제가 카드 형태로 표시됩니다.
3. **문제 카드를 클릭**하면, 우측에 유사한 문제가 유사도 순으로 표시됩니다.
4. 유사 문제를 **클릭**하면 원본 이미지와 상세 정보를 볼 수 있습니다.

> **유사도 뱃지 의미**:
> - 🟢 80% 이상 — 매우 유사한 유형
> - 🟡 60~79% — 관련 있는 유형
> - ⚪ 60% 미만 — 약한 관련성

## 학생 QnA 자동 매치업

학생이 **커뮤니티 > QnA**에 문제 사진을 첨부하면, AI가 자동으로 매치업 데이터베이스에서 유사한 문제를 검색합니다.

### 선생님 화면

1. **커뮤니티 > QnA** 탭에서 학생 질문을 클릭합니다.
2. 학생 질문 아래에 **"AI 매치업 결과"** 패널이 자동으로 표시됩니다.
3. 유사한 문제의 출처(강의 > 차시 > 시험), 문제 번호, 유사도를 확인할 수 있습니다.

> **참고**: 매치업 결과는 선생님에게만 표시됩니다. 학생에게는 보이지 않습니다.

## 지원 형식

| 형식 | 지원 여부 | 비고 |
|------|----------|------|
| PDF | ✅ | 스캔 시험지 권장 |
| PNG | ✅ | 고해상도 권장 |
| JPG/JPEG | ✅ | 고해상도 권장 |
| 한글 텍스트 | ✅ | Google Vision AI 한국어 지원 |
| 영어 텍스트 | ✅ | |
| 손글씨 | △ | 선명한 경우만 인식 |
| 수식 이미지 | △ | 텍스트 기반 수식만 인식 |

## 제한 사항

- 파일 크기: 최대 **50MB**
- 테넌트당 문서 수: 최대 **500개**
- 지원 형식: PDF, PNG, JPG만 가능

## FAQ

**Q: 문제 추출이 실패했어요.**
A: 이미지가 너무 흐리거나 회전되어 있으면 실패할 수 있습니다. 선명하게 다시 스캔해서 업로드해 주세요. 실패한 문서 옆의 🔄 버튼으로 재시도할 수 있습니다.

**Q: 유사 문제가 하나도 안 나와요.**
A: 매치업 데이터베이스에 비교할 문제가 부족합니다. 더 많은 시험지를 업로드할수록 유사 문제 추천 정확도가 높아집니다.

**Q: 시험 문제가 자동으로 등록되나요?**
A: 네. 시험 도메인에서 시험지를 업로드하면 자동으로 매치업 데이터베이스에 인덱싱됩니다.

**Q: 학생이 매치업 결과를 볼 수 있나요?**
A: 아니요. QnA 매치업 결과는 선생님 화면에서만 표시됩니다.
