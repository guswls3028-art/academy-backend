# V1.2.45 Release Notes

**Status:** Production-deployed and tenant 2 OMR exam 419 verified
**Release date:** 2026-06-09 KST
**Scope:** OMR manual subjective-score source of truth, item-score recompute, and no-placeholder essay score proof

This release closes the follow-up OMR scoring concern after v1.2.44: when an
exam has teacher-entered correct answers and custom subjective/essay point
values, those rows remain the scoring source of truth. If an essay question has
a positive teacher-entered point value but no `해설참조`/answer placeholder, the
essay max score is still exposed and used as part of the exam score shape.

## 1. Decision

OMR scoring for tenant 2 exam 419 remains **GO**.

The canonical score shape is:

- Questions 1-18: 3 points each.
- Questions 19-22: 4 points each.
- Question 23: 30-point essay/subjective question.
- Score shape: objective 70, subjective 30, total 100, source `sheet`.

## 2. Change Summary

Backend:

- Added a shared `manual_subjective_score` service for explicit manual
  subjective-score evidence.
- Result sync and item-score edits now preserve subjective score only when there
  is explicit evidence:
  - manual essay `ResultItem` rows,
  - latest `ResultFact(source="manual_subjective")`,
  - explicit manual-subjective attempt metadata.
- Item-score edits no longer treat `Result.total_score - Result.objective_score`
  as a hidden subjective score unless one of the above manual facts exists.
- Regression coverage now proves that a positive essay question score is
  exposed even when the answer key has no essay placeholder.
- Regression coverage also proves that a user-entered essay answer, when
  present, is returned through the admin detail API.
