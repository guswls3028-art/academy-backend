## 5. Residual Risk

- Conservative source-transfer is ready for use, but native HWP exact-layout
  reconstruction is still not guaranteed for every HWP/HWPX source. The current
  package intentionally includes review workbooks, preview text, embedded page
  images, manifests, and OCR candidates so teachers can correct safely.
- Beta rewrite remains beta because it can change problem semantics and must
  stay behind explicit user choice and teacher review.
- API steady state is intentionally one instance for cost. Burst/load and deploy
  safety depend on target tracking and temporary deploy headroom.
