# V1.2.28 Release Notes

**Status:** production-deployed
**Release date:** 2026-06-02 KST
**Scope:** Tenant 2 OMR fact fallback + zero-score custom sheet hardening

This release closes the repeated Tenant 2 OMR failure class where recognition
facts existed but score/result publication could still break for custom OMR
sheets and manual essay scoring.

## 1. Root Cause

- Tenant 2 exam `333` is a custom mixed sheet:
  - `choice_count=20`
  - `essay_count=5`
  - `max_score=100`
  - all `ExamQuestion.score` values are `0`.
- The OMR worker recognized the latest real submissions correctly:
  - `20/20 ok`
  - marker homography alignment
  - confirmed student matches for submissions `283` to `286`.
- The score publication path still had two structural gaps:
  - re-sync expected `SubmissionAnswer` rows even when durable
    `OMRDetectedAnswer` facts were available.
  - mixed sheets with zero raw question scores could infer objective max as
    `0`, so re-sync risked wiping objective ResultItem scores.
- Manual essay ResultItems also needed the same score-shape fallback so teacher
  item-score entry works when the sheet uses decorative essay slots without
  explicit per-question scores.

## 2. Fix Summary

- Added a canonical submission-answer map helper for result grading/sync.
- OMR re-sync now falls back to the latest durable recognition facts when
  `SubmissionAnswer` rows are missing.
- Existing non-zero OMR results are protected from incomplete fact/answer maps.
- Kept the new OMR fact fallback inside the refactor-boundary guard.
- `get_exam_score_shape()` now infers equal per-question scores when all
  component scores are zero but the exam has a positive max score.
  - For the real Tenant 2 shape this resolves to objective `80`, subjective
    `20`, total `100`.
- Manual item-score entry uses the same shape fallback:
  - zero-score essay items get a usable max score,
  - manual essay item scores compose with OMR objective scores,
  - automatic essay ResultItems from OMR sync are removed while manual essay
    items are preserved.

## 4. Operational Note

The repeated teacher-visible symptom was "OMR is not recognizing", but the
latest production evidence shows recognition was healthy. The broken class was
downstream result materialization for custom mixed sheets, OMR fact recovery,
and manual essay score composition.

The low scores on the real Tenant 2 submissions are consistent with the
recognized answers and the registered answer key. The displayed Result total is
the objective OMR score plus the manual essay score; `ExamResult` remains the
submission-level objective grading record.
