# V1.2.25 Release Notes

**Status:** production-deployed
**Release date:** 2026-06-02 KST
**Scope:** Tenant 2 OMR manual placeholder composition + production canary

This release closes the Tenant 2 OMR incident where recognition succeeded but
score publication could still appear broken after a teacher had already created
a manual score placeholder.

## 1. Root Cause

- Recent Tenant 2 OMR scans were recognized correctly:
  - `choice_count=20`
  - `essay_count=5`
  - recognition fact `20/20 ok`
  - marker homography alignment succeeded.
- One real submission was blocked because a prior manual `0` score had created
  an `ExamAttempt(submission_id=0)` for the same exam/enrollment.
- Since the exam allowed only one attempt, that placeholder consumed attempt 1
  and prevented the real OMR submission from becoming the representative
  attempt.
- The OMR result sync also had a hidden component-score risk: it could
  materialize essay questions as automatic `0` score ResultItems, making later
  manual essay scoring less stable.

## 2. Fix Summary

- Added a canonical `ExamAttemptService` path that attaches safe manual
  placeholders to the real OMR submission instead of creating a competing
  attempt.
- Manual total/objective placeholders attach only when their score is `0`.
- Manual subjective scores are component scores, so they can attach and be
  preserved even when non-zero.
- OMR Result sync now updates objective questions only.
- OMR Result sync preserves the existing manual subjective component for the
  same representative attempt.
- Stale automatic essay ResultItems with source `online`/`omr` are removed
  during sync; manual essay ResultItems are preserved.

## 4. Operational Note

The reported "OMR is not recognizing" symptom was real from the teacher's
perspective, but the worker recognition itself was healthy. The failure was in
the downstream score publication path after a manual placeholder had already
occupied the only allowed attempt.

For the currently inspected Tenant 2 submissions, the low objective scores are
the actual objective grading results from the recognized OMR answers. Published
Result totals may be higher than ExamResult objective totals because manual
subjective scores are now preserved and composed into the final displayed score.
