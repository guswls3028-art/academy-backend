# 설계 (인프라·워커)

**Status:** [VERIFIED] executable-entry summary, checked 2026-05-25 KST.
배포·워커 실행 진실은 `.github/workflows/v1-build-and-push-latest.yml`,
`scripts/v1/deploy.ps1`, `docs/infrastructure/deployment-architecture.md`,
`docs/operations/deployment-modes.md`를 우선한다.

---

## 1. 구성도

```
Internet → Cloudflare CDN
              ├── Frontend (Static)
              └── API (Django/Gunicorn)
                    ├── RDS PostgreSQL
                    ├── R2 (academy-ai, academy-video, academy-excel, academy-storage)
                    ├── SQS
                    │     ├── academy-v1-ai-queue           → AI Worker CPU ASG
                    │     └── academy-v1-messaging-queue    → Messaging Worker ASG
                    └── AWS Batch
                          └── academy-v1-video-batch-queue  → Video Worker job
```

---

## 2. 설정·코드

| 용도 | 경로 |
|------|------|
| API | `apps/api/config/settings/base.py`, `prod.py` |
| Worker | `apps/api/config/settings/worker.py` |
| SQS / Batch / ASG | `scripts/v1/resources/*.ps1`, `.github/workflows/v1-build-and-push-latest.yml` |

---

## 3. 워커 실행 경로

- **Messaging**: SQS → CloudWatch/ASG Target Tracking → `academy-v1-messaging-worker-asg`.
- **AI**: SQS → CloudWatch/ASG Target Tracking → `academy-v1-ai-worker-asg`.
- **Video**: AWS Batch 전용. daemon/SQS/Video ASG 경로는 2026-05-10 폐기.
- **배포**: backend `main` push → GitHub Actions worker refresh. 인프라 정합화는 `scripts/v1/deploy.ps1`.

---

## 4. 비용 (3~4 TB 가정, 월)

| 구분 | 낮음 | 높음 |
|------|------|------|
| EC2 Video 워커 | $5 | $20 |
| EBS | $3 | $8 |
| R2 저장 3~4 TB | 약 $45 | 약 $60 |
| **합계** | **약 $53** | **약 $93** |

R2 + Cloudflare CDN → egress GB 과금 없음. 시청 증가 = R2 Class B 요청만.

---

## 5. R2·큐 요약

- **R2 버킷**: R2_AI_BUCKET, R2_VIDEO_BUCKET, R2_EXCEL_BUCKET, R2_STORAGE_BUCKET, R2_ADMIN_BUCKET(academy-admin, 테넌트 로고), R2_ADMIN_PUBLIC_BASE_URL (base.py)
- **SQS**: `academy-v1-ai-queue`, `academy-v1-messaging-queue` + 각 DLQ
- **Batch**: `academy-v1-video-batch-queue`, `academy-v1-video-batch-jobdef`
- **워커 진입**: `apps/worker/video_worker/batch_main.py`, `apps/worker/messaging_worker/sqs_main.py`, `apps/worker/ai_worker/sqs_main_cpu.py`
