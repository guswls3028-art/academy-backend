# scripts/archive — v4 제외 레거시 스크립트 (실행 금지)

이 폴더는 **참고·비교용**이다. **실행하지 마세요.** 정식은 `scripts/v4/` 만 사용한다.

| 하위 | 설명 |
|------|------|
| **infra/** | 구 인프라 (Batch, EventBridge, SSM, CloudWatch 등). v4/templates에 복제됨. |
| **legacy/** | 구 scripts 루트의 .ps1, .py, .sh, .bat 등. |
| **redeploy/** | 구 redeploy 관련. |
| **scripts_v3/** | 구 scripts_v3 풀스택 배포. v4 deploy로 대체됨. |

파일은 제거하지 않고 보관한다.
